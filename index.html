Oráculo de dados

Projeto de Web Scraping Inteligente desenvolvido para monitoramento e extração automatizada de conteúdos jornalísticos em tempo real, estruturado com um pipeline robusto que abrange desde o carregamento e análise da estrutura HTML até o parsing e persistência dos dados extraídos. Utiliza logging avançado, controle de duplicação de entradas e suporte à extração de palavras-chave para análise semântica.
