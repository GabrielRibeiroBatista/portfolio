Oráculo de dados

Projeto de Web Scraping Inteligente desenvolvido para monitoramento e extração automatizada de conteúdos jornalísticos em tempo real, estruturado com um pipeline robusto que abrange desde o carregamento e análise da estrutura HTML até o parsing e persistência dos dados extraídos. O sistema é altamente modular, com suporte tanto a scraping via requests + BeautifulSoup quanto via Selenium para lidar com páginas dinâmicas e restrições como bloqueios por 403 Forbidden. Utiliza logging avançado, controle de duplicação de entradas e suporte à extração de palavras-chave para análise semântica.
